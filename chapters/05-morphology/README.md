# Morphology

Nyrakai expands its lexicon through two complementary derivational systems, compounding, and sacred transformations.

- **Verbal Derivation** — suffixes that attach to verbs (action-based)
- **Nominal Derivation** — suffixes that attach to nouns (thing-based)

Everyday derivations are flowing and chantable, while sacred derivations (marked by **æ**) are compressed, harsh, and alien.

## Quick Reference

### Verbal Suffixes (attach to verbs)

| Affix | Function | Example |
|-------|----------|---------|
| **-ra** | Nominalizer (action→thing) | ƶōr→ƶōrra (burn→fire) |
| **-ƨu** | Agentive (action→doer) | ƶōr→ƶōrƨu (burn→burner) |
| **-țal** | Resultative (action→result) | ƶōr→ƶōrțal (burn→ash) |
| **-k^e** | Diminutive (small/lesser) | ƶōr→ƶōrk^e (burn→ember) |
| **-ñor** | Augmentative (great/intense) | ƶōr→ƶōrñor (burn→inferno) |
| **-ɛm** | Adjectivizer (→"like X") | raț→rațɛm (night→nightly) |

### Nominal Suffixes (attach to nouns)

| Affix | Function | Example |
|-------|----------|---------|
| **-raš** | Keeper/Guardian | na'ēr→na'ēr-raš (water→water-keeper) |
| **-bren** | Place/Domain | fœra→fœra-bren (earth→land) |
| **-țæl** | Essence/State | țræn→țræn-țæl (person→personhood) |
| **-šek** | Tool/Instrument | r'ōk→r'ōk-šek (death→weapon) |
| **-raun** | Collective/Lineage | țræn→țræn-raun (person→tribe) |
| **-k^æț** | Sacred Intensifier | na'ēr→na'ēr-k^æț (water→abyssal flood) |

## The æ Pattern

Suffixes containing the diphthong **æ** signal elevated or sacred register:
- **-țæl** (essence) vs **-țal** (resultative)
- **-k^æț** (sacred intensifier)

This phonological marker helps distinguish everyday from ritual speech.
