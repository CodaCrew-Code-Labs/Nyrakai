# Negation

Nyrakai forms negation using the prefix **za-** attached directly to the verb stem.

---

## The Negator Prefix: za-

| Element | Form | Function |
|---------|------|----------|
| **za-** | /zai-/ | Negates verbs and adjectives |

The prefix **za-** derives from the standalone adverb **zæ** (not), creating a consistent negation system.

---

## Basic Negation

The prefix attaches directly to the verb stem, before any aspect markers:

**Affirmative:**
```
O + Vs + S + Va
[object] [verb] [subject] [aspect]
```

**Negative:**
```
O + za-Vs + S + Va
[object] [NEG-verb] [subject] [aspect]
```

---

## Examples

### Verbs

| Affirmative | Negative | Meaning |
|-------------|----------|---------|
| šōr | zašōr | know → not know |
| yɛn | zayɛn | see → not see |
| lem | zalem | hear → not hear |
| kōm | zakōm | come → not come |
| tal | zatal | say → not say |

### Adjectives

| Affirmative | Negative | Meaning |
|-------------|----------|---------|
| sem | zasem | good → not good (bad) |
| grōm | zagrōm | big → not big |
| n'æ | zan'æ | true → not true (false) |

---

## Negation in Sentences

**"I do not know"**
```
∅ + zašōr + fā + [aspect]
    NEG-know + I
```

**"She does not see the star"**
```
hīn + zayɛn + šā + [aspect]
star + NEG-see + she
```

---

## Double Negation

Nyrakai does **not** use double negatives for emphasis. A single **za-** is sufficient:

- ✅ `zašōr` = "do not know"
- ❌ `zazašōr` = ungrammatical

---

## Negation vs. Reversal

Note the distinction between **za-** (negation) and **fē-** (reversal):

| Prefix | Function | Example |
|--------|----------|---------|
| **za-** | Negates (not X) | zaƶōr = "not burn" (doesn't burn) |
| **fē-** | Reverses (undo X) | fēƶōr = "unburn" (extinguish) |

---

## Notes

- The **za-** prefix works in both everyday and sacred registers
- When negating words beginning with **z-**, the prefix still applies: `za- + zæ → zazæ` (not not) — though semantically redundant
- The standalone adverb **zæ** can also appear for emphasis, but the prefix is the standard grammatical negation
